# iExpense

**iExpense** — это удобное приложение для управления личными финансами, разработанное на Swift с использованием SwiftUI. Приложение позволяет отслеживать доходы и расходы, сортировать их по категориям, а также предоставляет визуализацию данных с помощью графикоов.

## Основные функции

- Добавление и редактирование записей расходов.
- Возможность разделения расходов по категориям для удобного анализа.
- Поддержка различных валют.
- Визуализация расходов в виде графиков.
- Простой и удобный интерфейс.

## Требования

- iOS 15.0 и выше.
- Swift 5.5.
- Xcode 13 и выше.
